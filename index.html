Perguntas frequentes

"O que eu devo experar?"

O NodeSchool é um ambiente de aprendizado autodirigido, no qual você traz seu próprio laptop para aprender. A intenção é que os participantes trabalhem nos workshops Node de nodeschool.io. Mas se você tem um projeto pessoal no qual está trabalhando, faça-o! O objetivo do NodeSchool é ajudar as pessoas a explorar e aprender o Node. Haverá mentores presentes nos eventos para ajudá-lo quando você encontrar obstáculos.

"Como eu instalo o Node? "

Nossa recomendação é instalar Node atráves de algo chamado"Homebrew".

1. Instale Homebrew copiando o seguinte comando no terminal/bash prompt:

$ ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"

2. Uma vez instalado, executebrew install nodeque deve instalar a versão mais recente do Node.

Se você tiver algum problema ao instalar o Homebrew ou o Node, não se preocupe. Nós faremos nosso melhor para ajudar você a se preparar assim que você chegar no evento.

"O que é um Workshopper?"

Workshopper é o nome usado para os módulos de lição de código aberto associados ao NodeSchool. Todos são auto-guiados (você não precisa participar de um workshop para fazer um) e a maioria trabalha offline.

"Como executo um Workshopper?"

Para começar com um tutorial do Workshopper, você precisará instalar e, em seguida, executar esse programa no terminal/bash. Aqui está um exemplo de como instalar e executar o Beginner Node Workshopper:

$ npm install -g learnyounode
$ learnyounode
